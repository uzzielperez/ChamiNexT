AI Agents Starter

This starter demonstrates a tiny agent loop that can call mock tools.

Quick start:

```bash
cd courses/ai-agents-starter
npm install
npm run dev
```

Files
- `src/App.jsx` — simple UI that runs an agent loop against mock tools
- `src/tools.js` — mock tool implementations (search, calc, echo)

Notes
- This starter uses mock tools for offline demos. Swap in real APIs or server functions where needed.
